# Stroke_Predictor

Two machine learning models, Decision Tree and Random Forest, trained to predict whether a patient is susceptible to stroke based on 12 attributes

Dataset: Stroke Prediction Dataset obtained from Kaggle https://www.kaggle.com/fedesoriano/stroke-prediction-dataset?select=healthcare-dataset-stroke-data.csv

Dataset was loaded and cleaned, then split into training and testing sets (80:20)

Models were trained with cross-validation, and performance was analyzed using confusion matrices, both regular and normalized

DT Model: 100% train, 92% test accuracy

RF Model: 100% train, 95% accuracy
